## Overview

Build an intelligent invoice and document processing solution on AWS using serverless services, Amazon Textract, and an external AI model.

The workshop covers the complete implementation path:

- Project security and data foundation.
- Frontend authentication and document upload.
- AI processing and asynchronous document ingestion.
- Result storage, review APIs, observability, governance, and end-to-end testing.
