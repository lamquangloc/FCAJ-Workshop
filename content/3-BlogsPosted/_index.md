This section lists and introduces the blogs I have posted to the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - Building a Smart Retry Mechanism for Serverless Queue Consumers on AWS](3.1-Blog1/)
This blog presents how to build a smart retry mechanism for stateless queue consumers on AWS by combining Lambda, SQS, and EventBridge Scheduler to gracefully handle downstream service issues.

### [Blog 2 - Building a scalable user search layer on top of Amazon Cognito](3.2-Blog2/)
This blog introduces a solution to build an advanced user search system on top of Amazon Cognito using OpenSearch, DynamoDB, and Lambda to overcome default search limitations.

### [Blog 3 - Building a Multi-Account Patch Compliance Dashboard with Kiro Specs](3.3-Blog3/)
This blog shares how to build a secure patch compliance dashboard for multiple AWS accounts through the Kiro Specs process (Requirements -> Design -> Tasks) ensuring data security.

### [Blog 4 - Building a Scalable User Search Layer on Amazon Cognito](3.4-Blog4/)
This blog analyzes the benefits of decoupling authentication and search functions, utilizing event-driven architecture to sync user data to OpenSearch for large-scale systems.

### [Blog 5 - How ALS GeoAnalytics LITHOLENS™ Revolutionizes Core Logging Through ML on EKS](3.5-Blog5/)
This blog explains how the LITHOLENS platform applies Machine Learning and Computer Vision on a hybrid infrastructure using EKS to automate the geological core sample analysis process.
