### Week 7 Objectives:

* Understand and apply IAM Permission Boundaries for advanced security management.
* Build a complete Serverless Backend using AWS Lambda, Amazon S3, and DynamoDB.
* Develop and integrate a Frontend to interact with Serverless APIs via API Gateway.
* Monitor Serverless applications utilizing Amazon CloudWatch and AWS X-Ray.
* Design and draw the architecture for the group project.

### Week 7 Achievements:

* Tightened access control using IAM Permission Boundaries to prevent privilege escalation risks.
* Mastered Serverless architecture concepts. Successfully built a complete application workflow spanning from Frontend -> API Gateway -> Lambda -> S3/DynamoDB.
